#  JavaScript Temel Algoritma Pratikleri

Bu depo, JavaScript öğrenim sürecimde geliştirdiğim temel mantık ve algoritma çalışmalarını içermektedir.

##  İçerik
* **Kullanıcı Kayıt Sistemi:** Dizi (Array) ve Nesne (Object) yapıları kullanılarak giriş-çıkış ve yeni kayıt mantığı kurgulanmıştır.[cite: 3]
* **Restoran Sipariş Uygulaması:** `while` ve `for` döngüleri, koşullu ifadeler ve matematiksel hesaplamalar ile bir sipariş yönetim sistemi oluşturulmuştur.[cite: 1, 2]

##  Teknik Detaylar
* **Vanilla JavaScript (ES6+):** Saf JavaScript mantığı ile geliştirilmiştir.[cite: 1, 3]
* **DOM & Console:** Uygulamalar tarayıcı konsolu (`F12 > Console`) üzerinden interaktif bir şekilde çalışmaktadır.[cite: 1, 3]

##  Nasıl Çalıştırılır?
1. Repodaki herhangi bir `.html` dosyasını indirin.
2. Dosyayı tarayıcınızda (Chrome, Edge vb.) açın.
3. `F12` tuşuna basarak **Console** sekmesine gelin ve uygulamayı kullanmaya başlayın![cite: 1, 3]
